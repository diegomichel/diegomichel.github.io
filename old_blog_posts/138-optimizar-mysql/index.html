Sun, Apr4, 2010

Si tienes la necesidad de optimizar mysql puedes usar el siguiente programa: MySQLTunner http://blog.mysqltuner.com/download/ Lo descargas con

wget mysqltuner.ply lo ejecutas> perl mysqltuner.plTe preguntara el usuario de la base de datos, y el password de la base de datos. y te dara instrucciones de que variables configurar en el archivo my.cnf asi como consejos para tus aplicaciones que usen mysql.
